RN_FlatList

发表于 2017-07-28 | 分类于 ReactNative

FlatList

功能

高性能的简单列表组件，支持下面这些常用的功能
完全跨平台
支持水平布局模式。
行组件显示或隐藏时可配置回调事件。
支持单独的头部组件。
支持单独的尾部组件。
支持自定义行间分隔线。
支持下拉刷新。
支持上拉加载。
支持跳转到指定行（ScrollToIndex

分组/类/区（section），请使用

属性

FlatList属性
ItemSeparatorComponent ReactClass 行与行之间的分隔线组件
不会出现在第一行之前和最后一行之后
ListEmptyComponent ReactClass React.Element 列表为空时渲染该组件
可以是React Component, 也可以是一个render函数， 或者渲染好的element
ListFooterComponent ReactClass 尾部组件
ListHeaderComponent ReactClass 头部组件
columnWrapperStyle StyleObj 如果设置了多列布局（即将numColumns值设为大于1的整数），
则可以额外指定此样式作用在每行容器上
data Array data属性目前只支持普通数组
其他特殊数据结构，例如immutable数组，请直接使用更底层的VirtualizedList组件
extraData any
getItem
getItemCount
getItemLayout
horizontal boolean 设置为true则变为水平布局模式
initialNumToRender number
initialScrollIndex number
keyExtractor
legacyImplementation boolean 设置为true则使用旧的ListView的实现
numColumns number
onEndReached
onEndReachedThreshold number
onRefresh ?() => void
onViewableItemsChanged void 一个新的Item渲染或者隐藏 的时候调用此方法
refreshing boolean
renderItem
viewabilityConfig ViewabilityConfig

方法

FlatList方法
①scrollToEnd(params?: object) 滚动到底部
②scrollToIndex(params: object) 跳转到指定行
③scrollToItem(params: object)
④scrollToOffset(params: {animated?: ?boolean, offset: number}) 滚动到指定的偏移的位置
⑤recordInteraction()
