RN-Flexbox Layout

FlexboxLayout作用

React Native中使用flexbox规则来指定某个组件的子元素的布局。Flexbox可以在不同屏幕尺寸上提供一致的布局结构

样式属性

flexDirection、alignItems和 justifyContent

Flex Direction

在组件的style中指定flexDirection可以决定布局的主轴。子元素是应该沿着水平轴(row)方向排列，还是沿着竖直轴(column)方向排列呢？默认值是竖直轴(column)方向。

理解

1.flex container : 理解父层。容器。嵌套n个内层item
2.main direction: 主轴 与item的排列方式有关（图示按照行排，主轴横向）
3.cross direction: 交叉周
