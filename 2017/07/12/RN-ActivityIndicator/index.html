RN_ActivityIndicator

显示一个圆形的loading提示符号

属性Props

animating

是否要显示指示器，默认为true，表示显示

color

滚轮的前景颜色（默认为灰色）

hidesWhenStopped

在没有动画的时候，是否要隐藏指示器（默认为true）。

size

指示器的大小。small的高度为20，large为36
