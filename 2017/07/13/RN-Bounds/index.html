组件的高度和宽度决定了其在屏幕上显示的尺寸

指定宽高

最简单的给组件设定尺寸的方式就是在样式中指定固定的width和height。
React Native中的尺寸都是无单位的，表示的是与设备像素密度无关的逻辑像素点

🌰

Flex

理解为权重
组件能够撑满剩余空间的前提是其父容器的尺寸不为零
如果父容器既没有固定的width和height，也没有设定flex，则父容器的尺寸为零。其子组件如果使用了flex，也是无法显示的

🌰🌰
